# 技术栈

本项目当前采用前后端一体仓库结构，前端目录为 `fastnote/`，后端目录为 `backend/`。

## 前端

- **Vue 3**：前端框架
- **TypeScript**：类型检查与开发体验增强
- **Ionic 8**：跨平台 UI 组件库
- **Vue Router 4**：路由管理
- **Vite**：开发服务器与构建工具
- **UnoCSS**：原子化 CSS
- **Sass**：样式预处理
- **Vitest**：单元与集成测试
- **Cypress**：E2E 测试
- **Tauri**：桌面端打包
- **Capacitor**：移动端原生能力桥接

## 前端数据与编辑能力

- **Dexie.js**：IndexedDB 本地存储
- **PocketBase JavaScript SDK**：认证、数据、realtime、文件访问
- **Tiptap**：富文本编辑器
- **Axios**：补充 HTTP 客户端
- **nanoid**：唯一 ID 生成

## 后端

- **Go**：后端宿主语言
- **PocketBase**：后端框架与运行时

当前后端职责：

- 启动 PocketBase
- 挂载前端构建产物
- 预留 hooks 与 migrations 能力

当前不包含业务 routes 和业务 schema。

## 开发环境

- **Node.js**
- **npm**
- **Go**
- **Git**
