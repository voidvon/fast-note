# 核心功能模块

本文档描述当前项目的核心模块划分，采用“前端 FSD + 后端宿主”视角。

## 1. 笔记编辑与管理

主要目录：

- `fastnote/src/entities/note`
- `fastnote/src/features/note-editor`
- `fastnote/src/features/note-save`
- `fastnote/src/features/note-delete`
- `fastnote/src/features/note-move`
- `fastnote/src/widgets/note-list`
- `fastnote/src/widgets/editor`

主要职责：

- 笔记创建、编辑、保存、删除、移动
- 文件夹层级与列表展示
- 详情页编辑体验
- 编辑器内容管理与自动保存

## 2. 会话与认证

主要目录：

- `fastnote/src/processes/session`
- `fastnote/src/entities/auth`
- `fastnote/src/shared/api/pocketbase/auth.ts`
- `fastnote/src/shared/api/pocketbase/auth-service.ts`

主要职责：

- 登录与注册
- 登录态恢复
- 用户信息缓存
- 会话启动与释放

## 3. 同步与实时能力

主要目录：

- `fastnote/src/processes/sync-notes`
- `fastnote/src/shared/api/pocketbase/realtime-service.ts`
- `fastnote/src/shared/lib/storage/*`

主要职责：

- 本地数据与云端同步
- 冲突处理
- realtime 订阅
- 同步状态维护

## 4. 公开笔记访问

主要目录：

- `fastnote/src/processes/public-notes`
- `fastnote/src/entities/public-note`
- `fastnote/src/pages/user-public-notes`

主要职责：

- 公开用户数据初始化
- 公开笔记列表与详情展示
- 公开路由就绪

## 5. 路由与导航

主要目录：

- `fastnote/src/app/router`
- `fastnote/src/processes/navigation`

主要职责：

- 路由表装配
- 导航恢复
- 桌面/移动分支处理
- 公开路由与私有路由衔接

## 6. 本地存储与基础设施

主要目录：

- `fastnote/src/shared/lib/storage`
- `fastnote/src/shared/api/pocketbase`
- `fastnote/src/shared/lib/logger`
- `fastnote/src/shared/lib/error-handling`

主要职责：

- Dexie 本地持久化
- PocketBase API 适配
- 错误处理与日志
- 通用基础设施封装

## 7. 后端宿主

主要目录：

- `backend/main.go`
- `backend/internal/server/bootstrap`
- `backend/internal/server/hooks`
- `backend/migrations`

主要职责：

- 启动 PocketBase
- 托管前端静态资源
- 为后续 hooks 与 migrations 预留落点

说明：

- 当前后端是宿主层，不承载业务模块实现
