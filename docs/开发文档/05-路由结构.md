# 路由结构

本应用当前使用 Vue Router 4 + Ionic Vue Router，路由入口位于 `fastnote/src/app/router/`。

## 路由清单

- `/`：重定向到 `/home`
- `/home`：私有首页
- `/login`：登录页
- `/register`：注册页
- `/n/:id`：私有笔记详情页
- `/f/:pathMatch(.*)*`：私有文件夹页
- `/deleted`：回收站页
- `/:username`：公开用户首页
- `/:username/f/:pathMatch(.*)*`：公开文件夹页
- `/:username/n/:noteId`：公开笔记详情页

## 路由实现

路由表定义在 [routes.ts](/Users/yytest/Documents/projects/fast-note/fastnote/src/app/router/routes.ts)，路由器实例与守卫定义在 [index.ts](/Users/yytest/Documents/projects/fast-note/fastnote/src/app/router/index.ts)。

```ts
import type { RouteRecordRaw } from 'vue-router'
import HomePage from '@/pages/home'

export const appRoutes: Array<RouteRecordRaw> = [
  { path: '/', redirect: '/home' },
  { path: '/home', name: 'Home', component: HomePage },
  { path: '/n/:id', component: () => import('@/pages/note-detail') },
  { path: '/login', name: 'Login', component: () => import('@/pages/login') },
  { path: '/register', name: 'Register', component: () => import('@/pages/register') },
  { path: '/:username/f/:pathMatch(.*)*', name: 'UserFolder', component: () => import('@/pages/folder') },
  { path: '/:username/n/:noteId', name: 'UserNote', component: () => import('@/pages/note-detail') },
  { path: '/:username', name: 'UserHome', component: () => import('@/pages/user-public-notes') },
  { path: '/f/:pathMatch(.*)*', name: 'Folder', component: () => import('@/pages/folder') },
  { path: '/deleted', component: () => import('@/pages/deleted') },
]
```

## 守卫与预处理

全局守卫负责两类前置逻辑：

1. 桌面端公开笔记详情路由重定向
2. 公开笔记路由的数据就绪

```ts
router.beforeEach(async (to) => {
  if (shouldRedirectDesktopNoteRoute(to.path, window.innerWidth)) {
    return '/home'
  }

  await ensurePublicNotesRouteReady(to)
  return true
})
```

相关流程位于：

- `fastnote/src/processes/navigation/*`
- `fastnote/src/processes/public-notes/*`

## 路由参数

当前主要参数如下：

- `/n/:id`
  - `id`：私有笔记 ID
- `/f/:pathMatch(.*)*`
  - `pathMatch`：私有文件夹路径
- `/:username`
  - `username`：公开用户标识
- `/:username/n/:noteId`
  - `noteId`：公开笔记 ID
- `/:username/f/:pathMatch(.*)*`
  - `pathMatch`：公开文件夹路径

在页面中通过 `useRoute()` 读取参数：

```ts
import { useRoute } from 'vue-router'

const route = useRoute()
const noteId = route.params.id as string
const username = route.params.username as string | undefined
```

## 路由分层约束

- 路由表与守卫只放在 `fastnote/src/app/router`
- 导航恢复、返回策略、桌面分栏路由选择放在 `fastnote/src/processes/navigation`
- 公开访问初始化放在 `fastnote/src/processes/public-notes`
- 页面组件只负责参数适配与页面装配，不直接承载守卫逻辑
