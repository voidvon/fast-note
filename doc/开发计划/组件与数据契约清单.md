## 5. 数据层契约（Database / PocketBase）

### 5.1 本地数据库契约（Dexie）

| 表名       | 字段                                                                                                   | 索引                                                    | 变更类型（新增/修改/删除）        | 兼容策略                                      |
| ---------- | ------------------------------------------------------------------------------------------------------ | ------------------------------------------------------- | --------------------------------- | --------------------------------------------- |
| notes      | `id,title,summary,content,item_type,parent_id,updated,is_deleted,is_locked,is_public,note_count,files` | `&id`, `[item_type+parent_id+is_deleted]`, `updated` 等 | 计划新增：无（保持兼容）          | 继续使用字符串 `id`，时间字段统一 `getTime()` |
| note_files | `hash,file,fileName,fileSize,fileType,created,updated`                                                 | `&hash`                                                 | 计划新增：`[待确认] upload_state` | 若新增状态字段，默认值向后兼容                |
| user_info  | `id,username,name`                                                                                     | `&id`                                                   | 无                                | 无                                            |
| metadata   | `key,value`                                                                                            | `&key`                                                  | 无                                | 无                                            |

### 5.2 云端契约（PocketBase）

| Collection   | 字段                                                       | 过滤规则                                      | 写入策略                                  | 冲突策略                             |
| ------------ | ---------------------------------------------------------- | --------------------------------------------- | ----------------------------------------- | ------------------------------------ |
| notes        | `id,user_id,updated,is_deleted,is_public,files,content...` | 私有同步：`user_id=auth.id && updated>cursor` | `updateNote` upsert（create/update 回退） | 比较 `updated`，新者覆盖旧者         |
| users        | `email,username,password...`                               | 认证接口使用                                  | 注册后自动登录                            | 认证失败清理 authStore               |
| public_users | `username,id,...`                                          | `username = "..."`                            | 只读查询                                  | 无写回                               |
| files        | `hash,path,type,size...`                                   | 按 hash 查询                                  | 文件上传后 URL 回填                       | 与 note files 引用一致性依赖内容替换 |

## 6. 路由契约（Router）

| 路由                           | 进入条件             | 参数                 | 权限                 | 失败回退                    |
| ------------------------------ | -------------------- | -------------------- | -------------------- | --------------------------- |
| `/home`                        | 应用启动或桌面重定向 | 无                   | 登录可选（离线可用） | 初始化失败仍挂载应用        |
| `/n/:id`                       | 选择私有笔记         | `id`                 | 私有上下文可编辑     | 桌面端重定向到 `/home`      |
| `/f/:pathMatch(.*)*`           | 进入私有文件夹       | `pathMatch`          | 私有上下文可编辑     | 桌面端重定向到 `/home`      |
| `/:username`                   | 访问公开主页         | `username`           | 无需登录             | 初始化失败显示错误态 + 重试 |
| `/:username/f/:pathMatch(.*)*` | 访问公开文件夹       | `username,pathMatch` | 无需登录             | 回退公开主页                |
| `/:username/n/:noteId`         | 访问公开笔记         | `username,noteId`    | 无需登录（只读）     | 回退公开文件夹/主页         |
| `/deleted`                     | 进入回收站           | 无                   | 私有上下文           | 回退 `/home`                |
| `/login`, `/register`          | 认证入口             | 无                   | 未登录访问           | 登录后回上一路径            |
